# Project conventions

## Git remotes

- `origin` → `sangxtg/tabernam-site` (personal fork)
- `upstream` → `8seneca-hub/tabernam-site` (canonical)

**Default push target is `upstream` (8seneca-hub).** When the user asks to "push", push to `upstream`, not `origin`. Do not push to the personal fork unless the user explicitly says so.
